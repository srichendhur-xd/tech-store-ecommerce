# Tech Store

Tech Store is an e-commerce platform designed for purchasing electronics, offering a seamless and user-friendly experience. Built with modern web technologies, it provides a responsive interface and optimized performance for users.

## Features

- **Browse Products:** View a wide range of electronic items.
- **Product Details:** Detailed specifications and features for each product.
- **Add to Cart:** Add and manage items in your cart easily.
- **Responsive Design:** Optimized for all devices.
- **Login Page:** Secure login to personalize your shopping experience.
- **Category Page:** Shop by categories to find products faster.

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **State Management:** Context API
- **Build Tool:** Vite
- **Version Control:** Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/tech-store.git
   ```

2. Navigate to the project directory:
   ```bash
   cd tech-store
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit:
   ```
   http://localhost:5173
   ```
